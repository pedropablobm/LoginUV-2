# 07. Historias de Usuario Iniciales

1. Como administrador, quiero ver equipos ocupados/libres por sede y sala para controlar disponibilidad en tiempo real.
2. Como operador, quiero crear usuarios manualmente o por CSV para habilitar acceso rápido.
3. Como estudiante/docente, quiero iniciar sesión en menos de 5 segundos para no perder clase.
4. Como administrador, quiero restringir a una sola sesión por usuario para evitar uso simultáneo no autorizado.
5. Como administrador, quiero marcar usuarios con multi-sesión permitida para casos especiales.
6. Como coordinación académica, quiero reportes PDF/XLSX por plan/semestre para control de asistencia.
7. Como soporte TI, quiero saber cuándo hubo apagado inesperado para diagnóstico de equipos.
